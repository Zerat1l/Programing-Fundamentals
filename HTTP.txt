Отговори за HTTP Quiz в Sli.do

1. Какво означава HTTP? - HyperText Transfer Protocol

2. За какво служи HTTP? - За трансфер на уеб-базирани ресурси

3. Какъв е принципа на HTTP? - Заявка-отговор

4. Как се нарича уеб-браузъра при HTTP комуникацията? - Клиент

5. Кои са четерите основни HTTP метода? - GET, POST, PUT, DELETE

6. Какво означава съкращенеито CRUD? - Create, Read, Update, Delete

7. Какъв тип заявка би произвела следния адрес: "www.nakov.com/?search=pesho"? - GET

8. За какво служи уеб формуляра при HTTP комуникацията? - За изпращане на заявка с данни към сървъра

9. Чрез какво изпращаме данни към сървъра от нашата уеб-страница? - Чрез уеб формуляр

10. За какво служи полето action на уеб формуляра? - За задаване на адреса, към който ще се изпратят данните

11. За какво служи метода post? - За изпращане на данни към сървъра

12. Как завършваме HTTP заявка? - С празен ред

13. Как се нарича елемента на заявката, в който изпращаме съдържанието? - Тяло на заявката (body)

14. За какво служат хедърите? - Дефиниция на оперативните параметри

15. Какво означава код 200? - OK

16. За какво служи хедъра Content-Type? - За указване формата на съдържанието

17. Какъв код връща сървъра при неоткрит ресурс? - 404

18. Какво се съдържа в елемента body на отговора? - Съдържанието, което клиента е заявил

19. Кой от следните адреси е валиден? - http://www.google.bg/search?q=http+get+vs+post

20. Какво означава URL? - Uniform Resource Locator

21. Кои са основните елементи на URL? - Протокол, хост, път

22. Какво е URL encoding? - Метод за представяне на запазени символи чрез поредица от специални символи
